# 🏨 Hotel Analytics Pipeline

An end-to-end Data Analytics + Web Scraping project that extracts hotel room data from Booking.com using Selenium and converts it into structured data for analysis.

## 📌 Features

- Hotel Name Extraction
- Room Type Extraction
- Price Scraping
- Breakfast Availability
- Excel Export

## 🛠️ Tech Stack

- Python
- Selenium
- Pandas
- Jupyter Notebook

## 📂 Project File

- SeleniumScraper final (1) (1).ipynb

## ⚙️ Installation

pip install selenium pandas webdriver-manager openpyxl

## ▶️ How to Run

1. Open Jupyter Notebook  
2. Run the notebook file  
3. Scraped hotel data will be generated
